# superstore_sales_analysis_Python

## Project Objective

The objective of this project is to analyze sales, profit, shipping time and product performance using the Superstore dataset. The analysis focuses on identifying trends, profitability, and business insights that can help decision making.

## Dataset

The dataset contains information about orders, sales, profit, shipping mode, regions, categories and products.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

* Sales by region
* Sales by segment
* Shipping time by ship mode
* Profit margin by category
* Top products by sales
* Sales vs Profit relationship

## Key Insights

* Technology category has the highest profit margin.
* Consumer segment generates the highest total sales.
* Standard Class has the longest shipping time.
* Some products generate very high sales and should be prioritized.

## Conclusion

This analysis helps understand sales performance, profitability and shipping efficiency. The results can help the company improve logistics, focus on profitable categories and prioritize high-performing products.
